Instructions on how to use this site—coming soon!
Known issues:
  • Participles are not currently supported (coming soon, though).
  • Verbs with prefixes (e.g. πάρειμι) are also not correctly handled. Some forms may work, but most will not (particularly those with augments).
  • Some verbs beginning with vowels may not be correctly augmented. I tried to get them right, but double check before you trust what you see.
  • And for that matter, double check the forms for all verbs you enter before trusting them. I worked pretty hard to get most things right, but there are certainly some things that are still wrong. If you see any problems, send an email to greekdrilling _at_ gmail _dot_ com.
  • If there are several possibilities for a single form, or more than one principle part for a single tense in a verb, I currently can only handle one of them. So, for instance, the sixth principle part of the verb βλάπτω can be either ἐβλάβην or ἐβλάφθην. You will have to pick one of the two and enter it, because I can't currently take both.
  • Sorting of words doesn't work quite right, because I don't handle the diacritics properly when sorting. This should be fixed pretty soon.
If you have comments, suggestions, or questions, email greekdrilling _at_ gmail _dot_ com. Thanks!
